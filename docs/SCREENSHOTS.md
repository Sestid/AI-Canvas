## 如何截图

### macOS
1. 全屏截图：`⌘ + Shift + 3`
2. 区域截图：`⌘ + Shift + 4`
3. 录制 GIF：使用 [Kap](https://getkap.co/) 或 [LICEcap](https://www.cockos.com/licecap/)

### Windows
1. 全屏截图：`Win + Print Screen`
2. 区域截图：`Win + Shift + S`
3. 录制 GIF：使用 [ScreenToGif](https://www.screentogif.com/)

### Linux
1. 使用 `gnome-screenshot` 或 `flameshot`
2. 录制 GIF：使用 [Peek](https://github.com/phw/peek)

## 保存位置

将所有截图保存到 `docs/images/` 目录下，文件名与上面列出的名称一致。

## 注意事项

- 确保截图内容清晰
- 使用深色主题
- 避免包含敏感信息
- GIF 文件大小控制在 5MB 以内
- 推荐使用 PNG 格式（体积小、质量高）
