# AI Canvas 使用指南

## 🎯 核心功能速览

### 1. 创建节点

**方法一：从左侧工具栏**
- 点击左侧工具栏中的任意节点图标
- 节点会自动创建在画布中心

**方法二：使用命令面板（推荐）**
- 按下 `⌘ + K` (Mac) 或 `Ctrl + K` (Windows)
- 输入节点名称或拼音首字母
- 使用 ↑↓ 键选择，Enter 确认

### 2. 编辑节点

**选择节点**
- 点击节点即可选中
- 选中后右侧属性面板会显示节点详情

**编辑属性**
- 在右侧属性面板中修改节点属性
- 或直接在节点卡片上编辑
- 所有修改会自动保存

### 3. 连接节点

- 拖拽节点顶部或底部的小圆点
- 连接到其他节点创建数据流

### 4. 画布操作

| 操作 | 说明 |
|------|------|
| **平移** | 拖拽空白区域 |
| **缩放** | 鼠标滚轮 |
| **多选** | 框选或 Shift + 点击 |
| **删除** | 选中后按 Delete 或 Backspace |

### 5. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘/Ctrl + K` | 命令面板 |
| `⌘/Ctrl + S` | 保存 |
| `Delete` | 删除节点 |
| `ESC` | 取消选择 |
| 右键 | 上下文菜单 |

## 📦 节点详解

### 文本节点 (Text Node)

**功能**：AI 文本生成或自定义输入

**使用场景**：
- 生成视频脚本
- 创建角色对话
- 输出文案内容

**操作步骤**：
1. 选择 "AI 生成" 或 "自定义输入" 标签
2. AI 模式：输入 Prompt → 选择模型 → 点击生成
3. 自定义模式：直接输入文本

---

### 图片节点 (Image Node)

**功能**：AI 图片生成

**使用场景**：
- 角色设计
- 场景生成
- 概念图创作

**操作步骤**：
1. 输入 Prompt 描述
2. 选择风格（写实/动漫/油画等）
3. 设置比例和分辨率
4. 调整机位参数（可选）
5. 点击生成

**高级功能**：
- 水平旋转：-180° ~ 180°
- 垂直角度：-90° ~ 90°
- 焦距：14mm ~ 200mm

---

### 视频节点 (Video Node)

**功能**：AI 视频生成

**两种模式**：
- **文/图生视频**：从文本或图片生成视频
- **首尾帧生视频**：上传首尾帧插值生成

**操作步骤**：
1. 选择生成模式
2. 输入 Prompt 或上传图片
3. 设置视频参数（比例、时长）
4. 选择运镜效果（可选）
5. 开启音效（可选）
6. 点击生成

**运镜选项**：
- 推进 / 拉远
- 左移 / 右移 / 上移 / 下移
- 环绕

---

### 音频节点 (Audio Node)

**功能**：AI 语音合成

**使用场景**：
- 配音生成
- 旁白制作
- 角色对话

**操作步骤**：
1. 输入台词文本
2. 使用工具栏插入停顿和语气词
3. 选择音色和语速
4. 点击生成

**技巧**：
- `[停顿]` - 插入停顿
- `[疑问]` - 疑问语气
- `[感叹]` - 感叹语气

---

### 脚本节点 (Script Node)

**功能**：AI 脚本生成

**使用场景**：
- 故事大纲
- 分镜脚本
- 剧情设计

**操作步骤**：
1. 输入 Prompt 描述需求
2. 选择 AI 模型
3. 点击生成
4. 复制结果使用

---

### 工具箱节点 (Tool Node)

**功能**：专业工具集

**工具类型**：

1. **角色三视图**
   - 生成角色正、侧、背三个视角
   - 用于角色设计参考

2. **多机位九宫格**
   - 生成 9 个不同机位
   - 用于分镜设计

3. **剧情推演四宫格**
   - 起承转合四格分镜
   - 用于剧情演绎

---

## 💡 最佳实践

### 工作流程建议

1. **从脚本开始**
   - 创建脚本节点生成故事大纲
   - 完善剧情结构

2. **角色和场景设计**
   - 使用图片节点生成角色和场景
   - 使用工具箱生成多角度参考

3. **生成素材**
   - 文本节点：台词和旁白
   - 图片节点：关键帧
   - 视频节点：动画片段
   - 音频节点：配音

4. **组织连接**
   - 用连线表示数据流
   - 建立清晰的工作流程

### 性能优化

- 定期保存（自动保存每 30 秒）
- 避免创建过多节点（建议 < 50 个）
- 大文件使用外部存储链接

### 协作技巧

- 使用统一命名规范
- 添加注释说明节点用途
- 定期导出备份（开发中）

## 🎨 界面说明

### 顶部工具栏

- **Logo + 项目名**：可点击编辑项目名称
- **保存状态**：显示最后保存时间
- **Undo/Redo**：撤销重做（开发中）
- **运行按钮**：批量运行节点（开发中）
- **导出按钮**：导出工作流（开发中）
- **设置**：全局设置

### 左侧工具栏

- 可折叠设计
- 悬停显示 Tooltip
- 点击创建对应节点

### 右侧属性面板

- 显示选中节点的详细信息
- 实时编辑节点属性
- 查看节点状态和历史

### 画布控制器

- **+/-** 按钮：缩放控制
- **适应视图**：自动调整到最佳视角
- **小地图**：快速导航

## 🔧 故障排查

### 节点无法创建？
- 检查是否点击了空白区域取消选择
- 尝试使用命令面板创建

### 连线无法建立？
- 确保拖拽的是节点上的连接点
- 检查节点类型是否兼容

### 属性修改不生效？
- 确认已选中节点
- 检查属性面板是否显示

### 画布卡顿？
- 减少节点数量
- 关闭不必要的连线
- 刷新页面重新加载
